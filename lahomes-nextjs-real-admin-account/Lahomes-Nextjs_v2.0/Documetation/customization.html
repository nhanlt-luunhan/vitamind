Customizing Sidebar Color, Topbar Color and Layouts

Larkon allows you to have customized left sidebar, overall layout width or right sidebar. You could turn the left sidebar to different size, theme (look) and size. You can customize it by specifying the layout class name on body element in your html. The config object properties accept the following values:

And for Customizing in existing themes, open the useLayoutContext.tsx file from src/context/useLayoutContext.tsx and change in config js it.

Title Type Options Description
theme String "light" | "dark" Changes overall color scheme to dark
topbarTheme String "light" | "dark" Changes the color scheme of the topbar to a given value
menu : Theme String "light" | "dark" Changes the color scheme of the menu to a given value
menu : size String "default" | "condensed" | "sm-hover" | "sm-hover-active" | "hidden" The left sidebar size is reduced to a condensed (small) size
