## Vì sao nên làm server mini?

Raspberry Pi đủ gọn, tiết kiệm điện và đủ mạnh cho các dịch vụ gia đình như lưu
trữ, media hoặc backup.

## Gợi ý dịch vụ dễ triển khai

- File server (SMB/NFS)
- Media server (Jellyfin/Plex)
- Backup tự động (rsync/restic)

## Khuyến nghị phần cứng

- SSD qua USB 3.0 để tăng tốc độ
- Nguồn ổn định, tốt nhất 5V/3A
- Tản nhiệt tốt nếu chạy 24/7
